# Maximiliano Garcia Mortigliengo – Resume

Personal portfolio and CV site for **Maximiliano Garcia Mortigliengo (Maxi)**, Senior Frontend Software Engineer.

**Live site:** [maxi-garcia-mortigliengo-cv.vercel.app](https://maxi-garcia-mortigliengo-cv.vercel.app)

## Features

- **CV & profile** — downloadable PDF, role, and social links
- **Work experience** — positions with timelines and descriptions
- **Projects** — featured side projects with repo and live links
- **Languages** — spoken language proficiency
- **Dark mode** — theme toggle persisted with Nanostores
- **AI assistant** — on-device chat powered by [WebLLM](https://webllm.mlc.ai/) (Llama 3.1 8B), with answers grounded in CV data

## Stack

- [Astro](https://astro.build/) 7
- [React](https://react.dev/) 19 (islands for the AI assistant)
- [Tailwind CSS](https://tailwindcss.com/) 4 (Vite plugin)
- [Nanostores](https://github.com/nanostores/nanostores) for theme and AI state
- [@mlc-ai/web-llm](https://github.com/mlc-ai/web-llm) for in-browser inference
- ESLint (`eslint-plugin-astro`, React rules for islands)
